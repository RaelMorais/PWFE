Variáveis em JS

Variáveis são espaços em memória para armazenar valores.

Tipos de variáveis

No JavaScript, as variáveis podem ser declaradas de várias maneiras.

Os padrões básicos do JS são Let Const Var.

O let é usado de forma genérica, recebendo int ou strings. 

                        
                          
let nome = "João"; //String

                          
let idade = 25; //Int

                          
let ativo = true; //Boolean

O const é usado para declarar variáveis imutaveis

Exemplos são números ou strings.

                                
                                   
const nome = "João";

                                   
console.log(nome);

O Array é um conjunto de dados.

Exemplos são números, strings e objetos. 

                            
                              
let numeros = [1, 2, 3, 4];

                              
console.log(numeros);

O Var é usado para declarar variáveis mutaveis.

Exemplos são: 

                            
                             
var nome = "João";

                              
var idade = 25;

                              
console.log(nome, idade);

Escopo é a forma como definiremos como iremos usar as variáveis criadas dentro de uma função.
