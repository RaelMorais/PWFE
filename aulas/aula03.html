Variáveis em JS

Variáveis são espaços em memória para armazenar valores.

Tipos de variaveis

No JavaScript, as variáveis podem ser declaradas de várias maneiras e podem armazenar diferentes tipos de dados.

Os padrões básicos do JS são Let Const Var.

O let é usado de forma genérica, podendo receber tanto valores inteiros como strings.

                   
                            
                              let nome = "João"; //String
                              let idade = 25; //Int
                              let ativo = true; //Boolean

O const é usado para declarar variáveis cujo valor não pode ser alterado.

Exemplos são números ou strings.

                                
                                   const nome = "João";
                                   console.log(nome);

O Array é um conjunto de dados que pode conter vários valores em uma única variável.

Exemplos são números, strings e objetos. 

                            
                              let numeros = [1, 2, 3, 4];
                              

                              console.log(numeros);

O Var é usado para declarar variáveis que podem ser alteradas durante a execução do código.

Exemplos de valores atribuídos a variáveis Var incluem strings, números, ou booleans. 

                            
                              var nome = "João";
                              

                              var idade = 25;
                              

                              console.log(nome, idade);

Escopo é a forma como definiremos como iremos usar as variáveis criadas dentro de uma função.
