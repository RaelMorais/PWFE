Funções

Função é um bloco de código que tem como preocupação a reutilização de códigos. A ideia principal é fazer valer a Modularidade e Simplicidade das características de um algoritmo.

Obs: Clean Code Não repetir códigos, usar nomes apropriados para variáveis e funções, seguir boas práticas de espaçamento e incluir documentação necessária para fácil manutenção.
