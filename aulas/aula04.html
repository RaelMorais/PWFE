Funções

Função bloco de codigo que tem como preocupação a reutilização de codigos. A ideia principal é fazer vale a Modularidade e simplicidade das características de um Algoritmo

Uma das formas de funções é usando função e/ou POO;

Estrutura de uma função em JavaScript:

NomeFunção (Atributos) {} Corpo da função

obs: Clean code de uma função: Não repetir códigos, as vaiaveis e funções devem ter nomes apropriados para o uso. Os espaçamentos devem seguir as boas práticas e trazer a documentaão necessaria para fácil manutenção.
